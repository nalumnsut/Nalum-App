Profile

Updates user profile. Accepts multipart/form-data. File upload is supported via 'profilePicture' field.

Parameters

No parameters

Request body

profilePicture
string($binary)

Profile picture file to upload (JPEG/PNG/WebP/GIF).

batch
string

Batch year.

branch
string

Branch name.

campus
string

Campus location.

city
string

Current city.

country
string

Current country.

currentCompany
string

Current company name.

currentRole
string

Current job role.

socialMedia
string

JSON object containing linkedin, github, twitter, website.

experiences
string

JSON array of experience objects.
